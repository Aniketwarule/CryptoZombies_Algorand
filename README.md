# 🧟‍♂️ AlgoZombies – Learn Algorand Smart Contracts by Building Games (Building for the AlgoBharat Hackseries 2.0)

AlgoZombies is an interactive, gamified platform (inspired by [CryptoZombies](https://cryptozombies.io/)) that teaches you how to build decentralized applications (dApps) and smart contracts on the **Algorand blockchain**.  

Instead of boring theory, you’ll **learn by coding** fun blockchain-powered games step by step.  

---

## ✨ Features  

- 🎮 **Gamified Learning** – Code your way through challenges and unlock new levels.  
- 🔗 **Algorand Focused** – Learn to write smart contracts in **PyTeal** and **Reach**.  
- 🛠️ **Hands-On Projects** – From simple token creation to full on-chain games.  
- 📚 **Beginner-Friendly** – No prior blockchain experience required.  
- 🌍 **Open Source** – Community-driven and free to learn.  

---

## 📚 What You’ll Learn  

- Basics of Algorand & its Pure Proof-of-Stake (PPoS) consensus.  
- Writing and deploying Algorand Smart Contracts (ASC1).  
- Using **PyTeal** and **Reach** for dApp development.  
- Building interactive blockchain-based games.  
- Connecting frontend apps to Algorand with **AlgoSDK**.
